# Sam - Neurodivergent Adaptations Tool

Web-based tool for adapting lesson plans for autistic pupils and pupils with PDA profiles.

## What's in This Package

- `index.html` - The web interface
- `vercel.json` - Configuration for Vercel
- `api/index.py` - Backend serverless function
- `api/requirements.txt` - Python dependencies
- `SETUP.md` - Complete setup instructions

## Quick Start

1. Upload all files to a new GitHub repository
2. Connect GitHub to Vercel
3. Add your Anthropic API key to Vercel
4. Done!

See `SETUP.md` for detailed step-by-step instructions.
